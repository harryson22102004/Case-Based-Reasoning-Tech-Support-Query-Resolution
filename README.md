# Case-Based Reasoning — Tech Support Query Resolution
Case-based reasoning system for intelligent IT support ticket resolution.
### ■ Research Highlights
- **Weighted Cosine Retrieval**: Represents cases as TF-IDF vectors and retrieves the $k$ most similar past - **Adaptation Heuristics**: Transforms retrieved solutions by substituting product names and version number- **Algorithms Focus**: Tracks case-base growth, forgetting, and retrieval accuracy over time, demonstrating **Tech Stack:** Python, `scikit-learn`, `NumPy`.
